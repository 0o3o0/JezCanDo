🤖 LinkedIn Message Automation

This project automates personalized outreach on LinkedIn using Python and Selenium. Key features include:

  • 📥 Reads an Excel file of LinkedIn profile URLs
  • 🔀 For each profile, selects one of four pre-defined message templates (rule-based)
  • ✉️ Sends the chosen message via LinkedIn’s messaging interface
  • 📝 Logs success/failure status for each profile
  • 📂 Exports results to a new CSV file with profile URL and message status
