🔎 LinkedIn Scraper Automation

This project is a LinkedIn automation tool I built using Python and Selenium. It performs the following actions:

  • 🔍 Searches for relevant profiles on LinkedIn based on specific keywords
  • 🤝 Sends automated connection requests
  • 📊 Tracks connection status for each profile
  • 📁 Exports the collected data + statuses into an Excel file

This project is especially useful for job seekers, marketers, or anyone wanting to generate leads on LinkedIn in a smart, automated way.
